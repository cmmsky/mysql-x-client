# mysql-x-client #


## 模块说明 ##
1. **core** 主要实现与MySQL server进行基本通讯，包括一些权限认证，查询和更新一类的功能。
1. **binlog** 在core的基础上实现binlog同步

## 说明 ##
代码仅是学习MySQL通讯协议的产物
